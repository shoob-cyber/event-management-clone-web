## 💡 HOW TO USE NEW FEATURES

### Use Parallax:
```html
<div data-parallax>Content</div>
```

### Use 3D Tilt:
```html
<div class="tilt-on-hover">Content</div>
```

### Use Stagger Animation:
```html
<div class="fade-in-up stagger-2">Content</div>
```

### Use Glow Effect:
```html
<div class="glow-gold">Content</div>
```

### Use Image Depth:
```html
<div class="image-depth overlay-dark">
  <img src="..." alt="...">
</div>
```

### Use Lightbox:
```html
<div class="lightbox-trigger">
  <img src="..." alt="...">
</div>
```
